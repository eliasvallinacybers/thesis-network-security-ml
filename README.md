# Thesis Network Security ML

This repository contains the materials related to my final thesis on network security and machine learning.

## Repository structure

- `tfg/`  
  Contains the final thesis document and the presentation slides.

- `python_code/`  
  Contains the Python code used in this project. It includes the code for the analysis of the CIC-IDS2018 dataset, as well as an additional script for feature selection and the techniques applied for that purpose.

- `generated_artifacts/`  
  Contains the artifacts generated from the Python scripts, including figures, tables, output files, results, and other generated resources.

## Notes

The repository is organized to keep the thesis documents, the implementation, and the generated outputs clearly separated and easy to navigate.
